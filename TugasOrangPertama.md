# Tugas Orang Pertama

**Role:** UI/UX Designer + Frontend Developer

**Penjelasan Role:**  
Kamu bertanggung jawab membuat tampilan aplikasi yang cantik, mudah digunakan, dan sesuai dengan blueprint desain yang sudah kamu buat.

**Tugas Utama & File yang Harus Dikerjakan:**

### 1. Layout Global (Paling Awal)
- `app/Views/layout/header.php` → Header atas (logo, tab navigasi)
- `app/Views/layout/sidebar.php` → Sidebar kiri (menu Dashboard, Tab 2, Tab 3, dll)
- `app/Views/layout/footer.php` → Footer bawah

### 2. Implementasi Halaman
- `app/Views/dashboard` → Tab 1 (Dashboard)
- `app/Views/responden` → Tab 2 (Data Responden)
- `app/Views/panduan` → Tab 5 (Panduan Metode)
- `app/Views/eksperimen` → **Tab 4 (Eksperimen Mandiri)** ← Paling penting

### 3. Frontend Interaktif
- `public/assets/js/eksperimen.js` → JavaScript untuk Tab 4 (tambah/hapus laptop, AJAX hitung ARAS)
- `public/assets/css/style.css` → Styling tambahan dengan Tailwind CSS

**Kegunaan Tugas Ini:**
- Membuat aplikasi terlihat profesional dan nyaman digunakan
- Memastikan user (termasuk dosen penguji) mudah memahami dan mengoperasikan sistem
- Tab 4 harus sangat interaktif karena ini fitur unggulan aplikasi

**Output yang Diharapkan:**
- Semua halaman tampil sesuai desain blueprint
- Kode frontend yang rapi dan mudah dibaca
